                  🔄 清除浏览器缓存指南
════════════════════════════════════════════════════════════════

✅ 文件已更新，但浏览器可能缓存了旧版本

📋 解决方案（按顺序尝试）：

方案1: 硬刷新（推荐）
─────────────────────────────────────────────────────────────
  Windows/Linux:
    • Chrome/Edge: Ctrl + Shift + R
    • Firefox: Ctrl + F5
  
  macOS:
    • Chrome/Edge: Cmd + Shift + R
    • Safari: Cmd + Option + R
    • Firefox: Cmd + Shift + R

方案2: 清除缓存后刷新
─────────────────────────────────────────────────────────────
  1. 打开开发者工具 (F12)
  2. 右键点击刷新按钮
  3. 选择"清空缓存并硬性重新加载"

方案3: 重启Flask服务器
─────────────────────────────────────────────────────────────
  1. 终止当前运行的服务器 (Ctrl + C)
  2. 重新运行: python web_main.py
  3. 刷新浏览器

方案4: 无痕/隐私模式
─────────────────────────────────────────────────────────────
  • Chrome: Ctrl/Cmd + Shift + N
  • Firefox: Ctrl/Cmd + Shift + P
  • Safari: Cmd + Shift + N

════════════════════════════════════════════════════════════════

✨ 已添加版本号强制刷新
─────────────────────────────────────────────────────────────
  CSS文件现在带有版本参数：
  • style.css?v=20251011
  • template_editor_v2.css?v=20251011
  
  这会强制浏览器重新加载CSS

─────────────────────────────────────────────────────────────
