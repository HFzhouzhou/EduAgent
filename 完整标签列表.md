## 💡 使用建议

### 1. 完整使用（所有72个标签）
适合需要详细教案的场景，包含思政元素和完整教学过程。

### 2. 选择性使用（部分标签）
根据实际需要选择相关标签，不用的环节可以不放标签或使用条件判断：

```jinja2
{% if practice_content %}
实践环节：{{practice_content}}
{% endif %}
```

### 3. 简化版（基础标签）
只使用基本信息、教学目标、重难点、教学过程主要环节即可。
